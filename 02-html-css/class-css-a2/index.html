¿Qué es la W3C?

El Consorcio WWW, en inglés: World Wide Web Consortium (W3C), es un consorcio internacional que genera recomendaciones y estándares que aseguran el crecimiento de la World Wide Web a largo plazo.

Los estándares W3C definen una plataforma web abierta para el desarrollo de aplicaciones.

W3C Developer Avenue es el lugar único para encontrar la información y las herramientas que los desarrolladores necesitan para aprender, construir y hacer avanzar la Web.

¿Qué es la WHATWG?

La plataforma en web en detalle: El Hypertext Application Technology Working Group, o WHATWG, es una comunidad de personas y empresas interesadas en la evolución de HTML y las tecnologías conexas.​.

El WHATWG trabaja en una serie de tecnologías que son partes fundamentales de la plataforma web.

También, trabajan en una serie de ideas que aspiran a convertirse en estándares algún día.
